1. What is DOM and what is the purpose of DOM?

Document object model is the representation of the html document as nodes and objects. The purpose of DOM is to manipulate the html element by using any programing language to make website dynamic amd interactive

2. How will you select HTML elements using DOM? Name the DOM methods.

To select html elements using DOM we have to use some DOM method. so some of the DOM methods are

  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Event bubble is a concept in the document object model . event bubble happens when element receives an event. and it bubble up to the parent and ancestor until it reach the root element of the dom tree

4. What is a callback function and why will you use it?

A callback function is a function that receives an argument from another function and is then called from inside the outer function to finish a task or action . The advantage of using a callback function is that you can make another function call after waiting for the outcome of a previous function call.
